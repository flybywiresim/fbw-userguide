# Primary Flight Display (PFD)

!!! info "Coming soon"
    This chapter is currently in development.

## Description

The PFD is responsible for many things. It displays many bits of data to the pilots such as airspeed, flight modes and more. There are many parts that make up the PFD. It contains an airspeed indicator, flight mode annunciators, vertical speed indicator, altitude indicator, artificial horizon, bank angle indicator, heading indicator and many other small features. 

Here you will find all the information about the PFD.
