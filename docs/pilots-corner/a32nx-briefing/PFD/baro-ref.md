# Barometric Reference

## Description

This is displayed under the altitude indicator and shows the barometric setting. 

If it is STD, the altimeter is in the standard setting. 

It will display the numerical setting in hectoPascals or inches of mercury. 

It will pulse if:

- STD is not selected by the pilot above the transition altitude in climb, or
- STD is still selected in approach below the transition level, or
- If a transition level has not been set, below 2500 feet radio altitude.
