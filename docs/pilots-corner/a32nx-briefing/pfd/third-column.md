# Lateral Mode Annunciators

## Description

These annunciators tell the pilots what lateral mode the Autopilot and the Flight Director are relying on and which modes are armed. 

---

### RWY

Displayed with green text. Displays when runway mode is engaged. This is only displayed if there is an active localiser signal. This mode provides lateral guidance during the takeoff roll up to 30 feet (radio). 

### RWY TRK

Displayed with green text. Displays when runway mode is engaged once airborne or above 30ft RA 

### HDG

Displayed with green text. Displays when heading mode is engaged. Heading mode is when the aircraft follows the FCU selected altitude (selected mode). 

### TRACK

Displayed with green text. Displays when track mode is engaged. Track mode is when the aircraft follows the FCU selected track (selected mode). 

### NAV

Green text: 

- Displays when NAV mode is engaged to guide the aircraft along the lateral flight plan (managed mode).

Blue text:

- Displays when NAV mode is armed. 

### LOC\*

Displayed with green text. Displays when localiser capture mode is engaged. 

### LOC 

Green text:

- Displays when localiser track mode is engaged. 

Blue Text:

- Displays when LOC mode is armed. 

### APP NAV

Green text:

- NAV mode is engaged during a non ILS approach. 

Blue Text:

- NAV mode is armed during a non ILS approach. 

### GA TRK

Displayed with green text. Displays when go around track mode is engaged.
